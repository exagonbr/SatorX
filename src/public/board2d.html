Do tabuleiro ao aprendizado profundo

        

          

            Em motores como o AlphaZero, a rede neural não “vê” madeira: recebe tensores derivados da posição
            (vários planos 8×8: peças, repetições etc.). Aqui você vê o mesmo estado em notação e uma grade pedagógica.
          

          
    
            
  • Estado s: posição legal (FEN ou "planos" abaixo) — entrada da rede.
    • 
            
  • Política π(a|s): probabilidades sobre lances legais — “para onde jogar”.
    • 
            
  • Valor V(s): expectativa de resultado a partir de s — “quão boa é a posição”.
    • 
            
  • Buffer de replay: pares (estado, ação, resultado) das suas partidas — conjunto para treino.
    • 
          

        

        

          

            Cada lance (com a opção acima) atualiza a rede TD; ao fim da partida o replay e as posições do buffer
            são gravados na base SQLite e o servidor agenda treino dos pesos heurísticos em segundo plano.
          

          

            npm run build-buffer reconstrói o buffer na BD e exporta data/replay_buffer.jsonl.
            npm run train-buffer treina a partir da BD; use npm run train-buffer-file só com o jsonl.
          

        

        

          Grade 8×8 (codificação simples para estudo)
          

            Letras = tipo de peça (maiúscula = branca). Em redes reais usam-se múltiplos canais binários por tipo e cor.
