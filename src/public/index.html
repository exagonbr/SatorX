Do tabuleiro ao deep learning

      

        

          Em motores como AlphaZero, a rede neural não “vê” madeira: recebe tensores derivados da posição
          (vários planos 8×8: peças, repetições, etc.). Aqui você vê o mesmo estado em notação e uma grade pedagógica.
        

        
    
          
  • Estado s: posição legal (FEN ou “planos” abaixo) — entrada da rede.
    • 
          
  • Policy π(a|s): probabilidades sobre lances legais — “para onde jogar”.
    • 
          
  • Value V(s): expectativa de resultado a partir de s — “quão boa é a posição”.
    • 
          
  • Replay buffer: pares (estado, ação, resultado) das suas partidas — dataset para treino.
    • 
        

      

      

        

          Neste projeto, após jogar na UI você pode agregar replays e ajustar pesos heurísticos:
        

        

          npm run build-buffer →
          npm run train-buffer -- 30 4 500
        

      

      

        Grade 8×8 (codificação simples para estudo)
        

          Letras = tipo de peça (maiúscula = branca). Em redes reais usam-se múltiplos canais binários por tipo e cor.
